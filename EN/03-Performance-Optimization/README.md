# Part 3: Comprehensive Performance Optimization

-----

Welcome to the heart of the course\! In this section, we learn the skills that distinguish a Senior Developer: the ability to diagnose, analyze, and fix database performance issues.

## Lesson List

* **[Lesson 1: Analyzing Slow Queries with `EXPLAIN`](01-Explain-Plan/README.md)**

  > Meet your best friend for diagnostics. Learn to read Execution Plans and spot bottlenecks.

* **[Lesson 2: Index-Level Optimization](02-Index-Level-Optimization/README.md)**

  > Applying advanced strategies like Cardinality, Covering Indexes, and Index Prefixes to build smart indexes.

* **[Lesson 3: Query-Level Optimization](03-Query-Level-Optimization/README.md)**

  > Avoiding common Anti-Patterns and learning techniques like Keyset Pagination.

* **[Lesson 4: Architecture-Level Optimization](04-Architecture-Level-Optimization/README.md)**

  > Looking at the big picture: Caching strategies and the impact of Data Types at scale.

* **[Lesson 5: Server-Level Optimization](05-Server-Level-Optimization/README.md)**

  > Configuring `my.cnf`, understanding `innodb_buffer_pool`, and using the Slow Query Log.
