# Part 6: Integration & Optimization in WordPress

-----

In this final section of the course, we apply our specialized MySQL knowledge to solve unique performance challenges of the world's most popular Content Management System: **WordPress**.

## Lesson List

* **[Lesson 1: Understanding WordPress Database Structure](01-Database-Structure/README.md)**

  > Getting familiar with key WordPress tables and its specific architecture, which is the source of many performance issues.

* **[Lesson 2: Solving the `wp_options` & `autoload` Issue](02-wp-options-autoload/README.md)**

  > Identifying one of the most common causes of WordPress slowness and fixing it with a simple query.

* **[Lesson 3: Writing Secure & Optimized Queries with `$wpdb`](03-wpdb-class/README.md)**

  > Learning how to use the core WordPress tool to write secure and optimized custom queries.

* **[Lesson 4: Understanding the EAV Problem in `wp_postmeta`](04-postmeta-eav-problem/README.md)**

  > Exploring one of the deepest architectural problems in WordPress and reviewing professional solutions (like custom tables).

* **[Lesson 5: Object Caching & Connecting WordPress to Redis](05-Object-Caching/README.md)**

  > Learning the most powerful optimization technique for WordPress: offloading the database using persistent caching.
